# Hotel Chat Bot — Руководство для владельца

> **Контакт разработчика:** [Имя / телефон / email — заполнить]

Бот автоматически отвечает на сообщения гостей в WhatsApp (и Telegram — в следующей версии). Написан на Python, работает на сервере 24/7.

---

## Что бот делает автоматически
- Отвечает на вопросы о ценах, заезде/выезде, удобствах, как добраться
- Собирает заявки на бронирование (имя, даты, количество гостей)
- Отправляет вам уведомление в WhatsApp, когда гость хочет забронировать
- Отвечает на кыргызском, если гость пишет по-кыргызски
- Отвечает на русском по умолчанию
- Помнит историю переписки с каждым гостем

## Что вы по-прежнему делаете сами
- Подтверждаете бронирование, связываясь с гостем напрямую
- Отправляете реквизиты для оплаты (бот их никогда не раскрывает)
- Отвечаете на вопросы, которые бот не может решить — он напишет "уточню у администратора"

## Как обновить информацию об отеле

Все данные отеля хранятся в одном файле — `system-prompt.txt` на сервере.

```bash
# Подключитесь к серверу
ssh hotelbot@ВАШ_IP

# Откройте файл
nano system-prompt.txt

# Замените текст в [СКОБКАХ] на реальные данные:
# [НАЗВАНИЕ ОТЕЛЯ], [ПОЛНЫЙ АДРЕС], [ЦЕНА], [КАК ДОБРАТЬСЯ] и т.д.

# Сохраните: Ctrl+O → Enter → Ctrl+X

# Перезапустите бот
systemctl restart hotel-chat-bot
```

Изменения вступят в силу немедленно — перезапуск занимает несколько секунд.

---

## Если бот перестал отвечать

```bash
# Подключитесь к серверу
ssh hotelbot@ВАШ_IP

# Проверьте статус
systemctl status hotel-chat-bot

# Посмотрите последние ошибки
journalctl -u hotel-chat-bot -n 50

# Перезапустите
systemctl restart hotel-chat-bot
```

Бот запускается автоматически при перезагрузке сервера — вручную вмешиваться обычно не нужно.
